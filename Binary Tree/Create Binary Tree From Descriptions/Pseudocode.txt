Pseudocode:
createBinaryTree(descriptions):

    map = value -> TreeNode
    children = set

    for each [parent, child, isLeft] in descriptions:

        if parent not in map:
            map[parent] = new TreeNode(parent)

        if child not in map:
            map[child] = new TreeNode(child)

        parentNode = map[parent]
        childNode = map[child]

        if isLeft == 1:
            parentNode.left = childNode
        else:
            parentNode.right = childNode

        add child to children

    for each value in map.keys():
        if value not in children:
            return map[value]      // root node

    return null

Java Code:
class Solution {
    public TreeNode createBinaryTree(int[][] descriptions) {
        Map<Integer, TreeNode> map = new HashMap<>();
        Set<Integer> children = new HashSet<>();

        for (int[] d : descriptions) {
            int parent = d[0];
            int child = d[1];
            int isLeft = d[2];

            map.putIfAbsent(parent, new TreeNode(parent));
            map.putIfAbsent(child, new TreeNode(child));

            TreeNode parentNode = map.get(parent);
            TreeNode childNode = map.get(child);

            if (isLeft == 1) {
                parentNode.left = childNode;
            } else {
                parentNode.right = childNode;
            }

            children.add(child);
        }

        for (int node : map.keySet()) {
            if (!children.contains(node)) {
                return map.get(node);
            }
        }

        return null;
    }
}
--------------------------------------------------------------------------
Time & Space Complexity:

Time Complexity: O(n), where n = descriptions.length, since we traverse the descriptions once and then scan all created nodes once.
Space Complexity: O(n) for the HashMap storing nodes and the HashSet storing child values.
